## Instalaci√≥n: 

```bash
# Actualice su lista de paquetes
$ sudo apt update

# directorio de alojamiento
$ sudo mkdir email

# Entre al repositorio
$ cd black_cover

# Permisos
$ sudo chmod 777 black_cover.sh

# Inicie Black Cover
$ sudo bash black_cover.sh

# Para Salir
$ Ctrl+c
```
